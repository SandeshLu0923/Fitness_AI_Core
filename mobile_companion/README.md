# Fitness AI Mobile Tracker Companion

The mobile companion should be a native Android app, not a browser tracker and not a Python/OpenCV desktop app.

Planned stack:

- Kotlin Android app
- MediaPipe Tasks Vision for Android
- On-device pose tracking
- Rep state machines for squats, push-ups, pull-ups, sit-ups, and jumping jacks
- Sync completed workout results to the Render backend

This folder is a placeholder for the Android project. An APK cannot be built from the current repository yet because no Android project or Gradle wrapper has been added.

Initial distribution can use a free GitHub Release APK download. Production distribution should use Google Play when available.
